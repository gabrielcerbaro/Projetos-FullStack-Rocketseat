## 📘 Sobre o Repositório

Este repositório reúne meus estudos, exercícios e projetos desenvolvidos durante minha jornada de aprendizado em Desenvolvimento Web Full-Stack com JavaScript.

Ao longo da formação, estou aprendendo desde os fundamentos da programação até a construção de aplicações completas utilizando:
